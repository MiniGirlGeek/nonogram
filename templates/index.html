What even is a Nonogram anyway?

A Nonogram is a puzzle that originated in Japan. They're logic based puzzles, that involve using clues to reveal a hidden picture. It comprises of a grid, which is where the picture will go, and you need to decide which cells need to be coloured, and which should remain blank. The numbers around the edges help you decide what to do.

How on earth do I go about solving one?

The numbers on the edge of a nonogram indicate the groups of coloured cells present in a row or column. Follow the tips below if you're not sure how to get started.

  • Tip 1

    Because each colum contains a single group of length 1, and the row has a single group of length 5, the entire row can be filled in.

  • Tip 2

    On a larger grid you can apply this method by filling in any full rows.

    If a full row lies on the edge of the grid you can 'push off' from the edge and complete the first group in that particular row or column.

    The next step is to mark crosses, you can do this either side of a completed group and on completed rows / columns

    Then simply fill out the rest.

  • Tip 3

    If a row / column isn't full, you can check to see if the groups overlap. Draw both extremes of the group arrangements, where a group overlaps with itself you know those cells have to be filled in.
